# 🎮 Tic! Tac! Toe! 게임

원하는 로고와 원하는 색상을 선택하여 빙고 게임을 해봅시다!

## [프로젝트 실행방법]

```
npm install
⬇️
npm start
```

## [Tech Stack]

- ✨ **개발 환경** : VSCode, Git, Github, Npm
- ✨ **주요 스택** : React, TypeScript
- ✨ **스타일링 도구** : Styled-Components
- ✨ **상태 관리** : Redux-Toolkit
- ✨ **기타** : ESLint, Prettier
